# Tech Stack
- **Language:** SWIFT 5
- **IDE:** Xcode 14
- **iOS:** 16
- **Archetecture:** MVC
- **Frameworks:**  UIKit, Foundation, Core Animation
- **Layout:** From Figma, Without Interface Builder only code
- **GCD:** Yes 
- **Features:** 
  - URLSession
  - DataModel
  - Rest API
  - Json decoding
  - TableView
  - CollectionView
  - Custom Navigation bar
  - Custom animation for cells
  - Custom fonts
  - Segues
  - And many more
